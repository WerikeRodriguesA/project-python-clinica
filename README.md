Este é um projeto prático desenvolvido em **Python** como trabalho acadêmico universitário. O sistema consiste em uma aplicação de terminal simples e intuitiva para o gerenciamento essencial de uma clínica médica, englobando o cadastro de pacientes, médicos e o agendamento de consultas.

---

## 🚀 Funcionalidades Principais

O sistema foi estruturado para resolver os fluxos básicos de atendimento de uma clínica:

* **👥 Cadastro de Pacientes:** Registro simplificado contendo dados como Nome, CPF e Telefone.
* **🩺 Cadastro de Médicos:** Registro dos profissionais de saúde com Nome, CRM e Especialidade.
* **📅 Agendamento de Consultas:** Vinculação de um paciente cadastrado a um médico disponível, definindo Data e Horário para o atendimento.
* **📋 Visualização de Relatórios:** Listagem rápida de todas as consultas agendadas para facilitar o controle do dia.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python 3.x
* **Persistência de Dados:** Estruturas de dados nativas (`listas` e `dicionários`) ou persistência em arquivos de texto (`.txt` / `.json`) para manter os dados salvos entre as execuções.
